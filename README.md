## ⚽ FIFA World Cup Dashboard (Power BI) 
This project showcases an interactive Power BI dashboard built using FIFA World Cup historical data. It provides insights into tournament statistics, 
country participation, and performance trends over time.



## 🚀 Overview -
The dashboard analyzes World Cup data from 1920 to 2014, highlighting key metrics such as goals scored, matches played, qualified teams, and 
country-wise performance.

## 📊 Key Features -
1.	🌍 Attendance by Country and Year
       Visualizes global participation and attendance trends across different World Cups
2.	📈 Competition Statistics by Year
       Displays trends in goals scored, matches played, and qualified teams over time
3.	🏆 World Cup Winners
       Shows distribution of winning countries and their title counts
4.	🌎 Countries Qualified
       Highlights countries with the highest number of World Cup appearances
5.	🏟️ Host Countries Analysis
       Displays countries that have hosted the World Cup


## 🛠️ Tools & Technologies - 
1.	Power BI
2.	Power Query (Data Cleaning & Transformation)
3.	DAX (Measures & Calculations)


## 🎯 Insights - 
1.	Brazil has the highest number of World Cup wins and qualifications
2.	European and South American countries dominate tournament outcomes
3.	The number of matches and goals has increased over time
4.	Hosting is distributed across multiple regions, with some countries hosting multiple times
